# tenmile-tutoring

Design previews for Michelle (Ten Mile Tutoring). Three visual directions included:

- `v1/` — classic/bookish: deep navy, warm gold, serif-led trustworthiness.
- `v2/` — mountain modern: spruce green, alpenglow coral, clean ridgeline styling.
- `v3/` — warm/playful: sunrise palette, rounded shapes, friendly family tone.

Built as full tutoring landing pages with services, about, process, testimonials, and Netlify forms.

Preview locally:

```bash
python3 -m http.server 8000
```
